如何在行内代码中输入一个或多个反引号 `` ` ``？如何在代码块中显示 ` ``` `？

关键是**用不同数量的反引号包围之**。

<!--more-->

首先你要知道，*行内代码*是用少于 3 个（1 个或 2 个）反引号包围起来形成的，而*代码块*是用 3 个及以上反引号包围起来形成的。

## 在行内代码中输入反引号

在行内代码中，内容头尾两端多余的空格会被自动除去。利用这一特点，可以避免内容头尾两侧的反引号和作为边界的反引号“黏”在一起。

```xxx
- 一个反引号：`` ` ``
- 两个反引号：` `` `
- 三个反引号：` ``` ` 或 `` ``` ``
- 更多反引号：` ````...```` ` 或 `` ````...```` ``
```

↓↓↓

- 一个反引号：`` ` ``
- 两个反引号：` `` `
- 三个反引号：` ``` ` 或 `` ``` ``
- 更多反引号：` ````...```` ` 或 `` ````...```` ``
