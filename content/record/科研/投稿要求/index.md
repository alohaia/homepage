## 图片 {alias="Figures"}

参考自 [OUP journals--guidance for preparing artwork 中文](https://static.primary.prod.gcms.the-infra.com/static/site/journals/document/images-author-guidance.pdf?node=1bf05d0b2fbd9c529a23&version=490455:30c2211aa70bba63a5ee)。

{{% section %}}图像类型和文件格式{{% /section %}}

- 使用未压缩的 TIFF 格式保存栅格图像；JPG / PNG 格式也可以，但分辨率可能略低。
- 使用 SVG / EPS / PDF 格式保存包含矢量部分（如文本、绘图）的图像。
